# MinecraftWrapperJNI

JNI Cheat/Dll with ImGui overlay and runtime hooks.

## Features

- Fast Break
- Fast Place
- Auto Sprint
- Anti Knockback
- TriggerBot

## Controls

- `INSERT`: Toggle menu
- `DELETE`: Exit/unload

## Build

```powershell
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```
