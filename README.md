JNI Cheat/Dll with ImGui overlay and runtime hooks.
Works on 1.21.11 and 1.21.11 Fabric fabric-loader-0.18.4-1.21.11
## Features

- Fast Break
- Fast Place
- Auto Sprint
- Anti Knockback
- TriggerBot

## Controls

- `INSERT`: Toggle menu
- `DELETE`: Exit/unload

## Build

```powershell
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```
